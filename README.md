# 自动化设备运动控制上位机系统

## 项目信息
- **开发者**: lqf
- **开发时间**: 2026
- **技术栈**: C#, .NET Core 8, Windows Forms
- **硬件接口**: 运动控制卡（WJ_API.dll）

## 项目简介
本项目是一个基于运动控制卡的多轴联动自动化设备上位机控制软件，实现了对多自由度运动控制系统的精确控制。系统支持多种控制模式，包括点动控制、长动控制和自动轨迹规划，能够满足不同场景下的自动化加工需求。

## 主要功能

### 1. 多轴运动控制
- **X/Y/Z 直线轴控制**: 支持三轴独立运动和联动
- **旋转轴控制**: X轴摆动和Z轴自转控制
- **抛光轮控制**: 抛光轮转速调节

### 2. 三种控制模式
- **点动模式**: 按下按钮时轴运动，松开即停，适合精确定位
- **长动模式**: 点击按钮后轴持续运动，需手动停止
- **自动轨迹模式**: 
  - 行切法（栅格轨迹）：适用于平面抛光
  - 环切法：适用于圆形工件抛光

### 3. 速度参数配置
- 各轴独立速度设置（10-80 r/min 多档位可选）
- 转动轴角速度配置（rad/s）
- 实时速度切换

### 4. 安全保护机制
- **急停功能**: 所有轴独立急停按钮 + 全局急停
- **缓停功能**: 平滑减速停止，减少机械冲击
- **状态指示**: 实时显示各轴运行状态（红/绿灯指示）

### 5. 坐标系统
- 三维坐标实时显示
- 坐标原点设置与清除
- 当前位置可视化

## 技术亮点

1. **硬件抽象层封装**: 通过 P/Invoke 调用底层 C++ DLL，实现对运动控制卡的封装
2. **状态机设计**: 不同模式下 UI 控件状态的自动管理
3. **轨迹规划算法**: 实现往复扫描路径的自动生成和执行
4. **开环时序控制**: 基于时间片的运动序列控制逻辑
5. **参数化配置**: 支持动态调整距离、次数、直径等工艺参数

## 系统架构

```
用户界面层 (Windows Forms)
    ↓
业务逻辑层 (Form1.cs - 运动控制逻辑)
    ↓
硬件抽象层 (WJ_API.cs - P/Invoke 封装)
    ↓
运动控制卡 (WJ_API.dll)
    ↓
驱动器 → 电机 → 机械结构
```

## 应用场景
- 磁流体抛光加工
- 精密曲面抛光
- 自动化表面处理
- CNC 数控系统教学演示



## 注意事项
- 使用前请确认运动控制卡已正确连接
- 首次运行需要进行轴参数校准
- 急停按钮仅在连接成功后生效
